# Downloading a game

### 1. Download the lua file <sup><sub>[[What are the lua files for?](/faq)]</sub></sup>

1. Open [openlua.cloud](https://openlua.cloud)
2. Search for the game you want to add to your Steam Library for free
3. Click Download and complete the ad
4. After completing the ad, make sure to read any popups that might've appeared.

### 2. Open SteamTools and add the game

If you haven't already, open SteamTools and a gray floating Steam icon should appear.

<details>
<summary>If the floating icon did not appear</summary>

A white Steam icon should've appeared in your tray. Right click it, and select **Display Floating Window**.

![](images/652ce4c9-983a-4d0e-b3e7-d00b6366c7dd.png)

:::tip
Can't see it? Make sure to check the Hidden icons menu too:

![](images/749c9db7-d142-43b6-b3d2-a5d04222671b.png)
:::

</details>

After that, drag the lua file you've downloaded onto the floating Steam icon.

### Congrats!
If you've followed all of the steps correctly, you should now have the game in your Steam Library, available to download for free.
