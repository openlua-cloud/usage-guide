# EA SPORTS FC™ 26

### 1. Register a new Steam account
1. Download [UrbanVPN](https://www.urban-vpn.com/) and select Mexico, UAE or Colombia.
:::note
You don't have to use [UrbanVPN](https://www.urban-vpn.com/) specifically. You can use any other system-wide VPN that  allows selecting a server in Mexico, UAE or Colombia.
:::

2. [Register a new Steam account](https://store.steampowered.com/join/) and select Mexico, UAE or Colombia in the `Country of Residence` field.

:::important
Do not register the Steam account in the app. You have to specifically do it through [this page](https://store.steampowered.com/join/).
:::

3. Once you login with your newly registered Steam account in the app, disable the VPN.

:::note
You can uninstall the VPN at this point, as you won't need it for the rest of this guide.
:::

### 2. Download the [EA SPORTS FC™ 26 SHOWCASE](https://store.steampowered.com/app/3629260/) from Steam

### 3. Launch the game once so all the necessary files are created
:::important
Launch the game directly via it's .exe file, not through your Steam Library or the EA app.
:::

### 4. Patch the game's files
1. Open the game's folder ([How to open Steam game's folder?](/extras/opening_a_steam_games_folder))
2. [Download](https://i.sobakin.tech/api/files/download?objectName=cmh4y3oq40000pu09qh2eonah%2F1763932859563-r348jO2PNxsQ.zip) and open the .zip file with the patched files
3. Select all of the files and drag them into the game's folder
4. If a window as shown below pops up, click **Replace the file in the destination**. It may appear multiple times.

![](../../images/c3956f7a-c018-448b-9e02-973d28ed04c0-1.png)

5. Delete `FC26_Showcase.exe`
6. Rename `FC26_Showcase fixed.exe` to `FC26_Showcase.exe`
### 5. Generate the Denuvo token
1. Open "EA.Denuvo.Token.Dumper.exe" and press Start
:::note
Make sure to **uncheck** "Add DenuvoToken to anadius.cfg even if it exists"
:::

2. Copy the generated Denuvo token
3. Open `anadius.cfg` in the game folder
4. Find `DenuvoToken` (using CTRL + F) and replace the `PASTE_A_VALID_DENUVO_TOKEN_HERE` with your copied token
:::danger
Make sure you do not erase the quotes around the token!
:::

### Congrats!
If you've followed all of the steps correctly, you should now be able to play EA SPORTS FC™ 26.

:::important
Always launch the game directly via it's .exe file, not through your Steam Library or the EA app.
:::
