# ARC Raiders
### Prerequisites 
- You have downloaded ARC Raiders from [openlua.cloud](https://openlua.cloud) and installed it

### 1. Add Cairn Demo to your library
1. Open the [Cairn Demo Steam Store Page](https://store.steampowered.com/app/3170150/Cairn_Demo/)
2. Scroll down and click **Add to Library**

### 2. Create an Embark ID
1. Open the [Embark Sign-up page](https://id.embark.games/id/sign-in)
2. Click on Steam
3. Click Sign In
4. Complete the Embark ID creation process

### 3. Patch the game's files
1. Open the game's folder ([How to open Steam game's folder?](/extras/opening_a_steam_games_folder))
2. [Download](https://cdn.openlua.cloud/bypasses/ARC%20Raiders.rar) the patch files
3. Extract the contents of the .rar file (Password: openlua.cloud)

:::info IF YOU INSTALLED ARC RAIDERS IN A CUSTOM LOCATION (E.G., A DIFFERENT DRIVE)
1. Right-click `Launcher.bat` and select **Edit with Notepad**
2. Update the path to match your actual `PioneerGame.exe` location
3. Save the file
:::

4. Copy all extracted files into the game's folder
5. If prompted, click **Replace the file in the destination** (this may appear multiple times)

![](./images/c3956f7a-c018-448b-9e02-973d28ed04c0-1.png)

6. Launch the game using `Launcher.bat`

### Congrats!
If you've followed all of the steps correctly, you should now be able to play ARC Raiders.

:::note
You can play on official servers.
:::

:::important
Always launch the game using `Launcher.bat`
:::

:::danger
**DO NOT BUY ANY IN-GAME ITEMS WITH REAL MONEY!** It will most likely get your Embark ID banned.
:::
