Downloading a game

1. Download the .lua file​

  1. Open Manifestor.cc
  2. Search for the game you want to add to your Steam Library for free
  3. Click Download

2. Open SteamTools​

If you haven't already, open SteamTools and a gray floating Steam icon should appear.

If the floating icon didn't appear

A gray/white Steam icon should've appeared in your taskbar:

Right click it, and select Display Floating Window.

3. Drag the .lua file onto the floating icon​

Congrats!​

If you've followed all of the steps correctly, you should now have the game in your Steam Library, available to download for free.
